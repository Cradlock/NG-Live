# MainServer - backend часть для NG-Live

## Структура 
```

uploads <- Ресурсы которые загружаются во время работы приложения
│   └── ... 
│
│
src <- исходный код
└── main
    ├── java
    │   └── com
    │       └── nglive 
    │           │   ├── account <- пакет логики аккаунта
    │           │       ...
    │           │   ├── auth <- пакет пользовтеля
    │           │       ...
    │           │   ├── common <- общие функции
    │           │       ...
    │           │   ├── shared <- общий класс для всех остальных
    │           │       ...
    │           │   
    │           └──MainVerticle.java <- точка входа
    │ 
    │ 
    └── resources
        └── resources
            └── db <- Миграции
                └── migration 

```

## Зависимости
- `vertx`: Основной модуль. Быстрый ассинхроный event-loop сервер
