# MainServer - backend часть для NG-Live

## Структура 
```

uploads <- Ресурсы которые загружаются во время работы приложения
│   └── ... 
│
│
src <- исходный код
└── main
    ├── java
    │   └── com
    │       └── nglive
    │           ├── dto <- DTO модели
    │           │   └── ...
    │           │ 
    │           ├── entities <- Класс для работы с запросами в бд
    │           │   └── ...
    │           │ 
    │           ├── routers <- Маршруты
    │           │   └── ...
    │           │ 
    │           ├── services <- Бизнес логика
    │           │   └── ...
    │           │ 
    │           ├── verticles <- Основные вертикали
    │           │   └── ...
    │           │   
    │           └──MainVerticle.java <- точка входа
    │ 
    │ 
    └── resources
        └── resources
            └── db <- Миграции
                └── migration 

```

## Зависимости
- `vertx`: Основной модуль. Быстрый ассинхроный event-loop сервер
